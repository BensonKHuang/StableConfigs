RUNNING PYTHON FILES ON LINUX

To run in command line (in linux), you will need the following:

+ python3.7
+ pysat (python-sat)

To install necessary software: 

    # Have a python version of 3.7 or higher on your machine
        sudo add-apt-repository ppa:deadsnakes/ppa
        sudo apt-get install python3.7

    # Create a user environment or virtualenv
        python3.7 -m pip install python-sat --user 

To run tests, go to individual folder and run:

    # export PYTHONPATH={/path/to/StableConfigs}
        export PYTHONPATH=/home/bhuang/SeniorDesign/StableConfigs

    # Run individual unit tests
        python3.7 -m unittest -v SATSolverTest.py
        python3.7 -m unittest -v ParserTest.py
